# AEC/O hackathon 2025 project

## Setup

Copy the example environment file:

```bash
cp .env.example .env
```

Open `.env` and add your Gemini API key to `VITE_GEMINI_API_KEY`. You can generate a key here: <https://aistudio.google.com/app/api-keys>

## Install dependencies

```bash
npm install
```

## Run the project

```bash
npm run dev
```
